Clase 1: Introducción a los Redes de Datos II

Objetivos de la clase:

Comprender qué es un sistema operativo y sus funciones esenciales.

Identificar los diferentes tipos de Redes de Datos II y sus características distintivas.

Conocer la relevancia de los Redes de Datos II en el desarrollo de software.

¿Qué es un sistema operativo?:

Un sistema operativo (SO) es un software esencial que actúa como intermediario entre el hardware de una computadora y las aplicaciones o programas que se ejecutan en ella. Es el programa principal que gestiona los recursos del sistema y proporciona una interfaz para que los usuarios y las aplicaciones interactúen con la computadora.

En términos sencillos, un sistema operativo es como el "director" o el "maestro" de la computadora, que coordina y supervisa todas las actividades del sistema para que funcione de manera eficiente y sin problemas.

Funciones principales de un sistema operativo:

Gestión de recursos: El sistema operativo administra y asigna los recursos de hardware, como la CPU (Unidad Central de Procesamiento), la memoria, el disco duro, los dispositivos de entrada/salida (como teclados, ratones, impresoras, etc.), y otros componentes, de manera que múltiples programas puedan compartirlos y utilizarlos de manera segura y equitativa.

Interfaz de usuario: Proporciona una interfaz mediante la cual los usuarios pueden interactuar con la computadora. Esto puede ser a través de una interfaz gráfica de usuario (GUI) con ventanas, iconos, menús, etc., o mediante una interfaz de línea de comandos (CLI) donde los comandos se ingresan mediante texto.

Gestión de archivos y directorios: El sistema operativo se encarga de organizar y gestionar los archivos almacenados en el disco duro. Permite crear, copiar, mover, renombrar y eliminar archivos, así como organizarlos en diferentes carpetas o directorios.

Administración de procesos y tareas: Controla y supervisa los procesos y tareas que se están ejecutando en la computadora. Un proceso puede ser un programa en ejecución, y el sistema operativo se asegura de que cada proceso obtenga el tiempo de CPU necesario para realizar su trabajo.

Gestión de memoria: Controla el acceso a la memoria RAM (Random Access Memory) para garantizar que los programas tengan suficiente espacio para ejecutarse y compartir datos de manera segura sin interferir entre sí.

Gestión de dispositivos: Administra y controla los dispositivos de hardware conectados a la computadora, asegurándose de que funcionen correctamente y sean accesibles para las aplicaciones que los requieran.

En resumen, el sistema operativo es un componente fundamental de cualquier computadora moderna, ya que permite una interacción más amigable con el hardware y garantiza que los recursos se utilicen de manera eficiente para que las aplicaciones puedan ejecutarse sin problemas. Cada vez que enciendes tu computadora, el sistema operativo es lo primero que se carga y es el encargado de coordinar todas las operaciones necesarias para que puedas usarla de manera efectiva.

Tipos de Redes de Datos II:

Existen varios tipos de Redes de Datos II, cada uno diseñado para satisfacer diferentes necesidades y aplicaciones. A continuación, se presentan los principales tipos de Redes de Datos II:

Redes de Datos II de Escritorio

sistemas diseñados para computadoras de uso personal o estaciones de trabajo. Están orientados a brindar una interfaz gráfica de usuario (GUI) que facilite la interacción del usuario con la computadora. Algunos ejemplos de Redes de Datos II de escritorio son:

Microsoft Windows: Es uno de los Redes de Datos II más populares para computadoras personales.

macOS: El sistema operativo desarrollado por Apple para sus computadoras Mac.

Redes de Datos II Móviles:

Están diseñados para dispositivos móviles, como teléfonos inteligentes y tabletas. Son capaces de gestionar recursos limitados y optimizar el consumo de energía. Algunos ejemplos de Redes de Datos II móviles son:

Android: Desarrollado por Google y utilizado en la mayoría de los dispositivos Android.

iOS: El sistema operativo exclusivo de los dispositivos de Apple, como iPhone y iPad.

Redes de Datos II de Servidores:

Estos sistemas están diseñados para gestionar grandes volúmenes de datos y manejar múltiples solicitudes de clientes al mismo tiempo. Son utilizados en servidores para proporcionar servicios a través de redes o internet. Algunos ejemplos son:

Linux: Una amplia variedad de distribuciones de Linux se utilizan en servidores debido a su estabilidad y seguridad.

Windows Server: La versión del sistema operativo Windows diseñada para entornos de servidor.

Redes de Datos II Embebidos:

Están integrados en dispositivos y equipos especializados, como electrodomésticos, sistemas de entretenimiento, dispositivos médicos, entre otros. Estos sistemas suelen ser altamente optimizados y tienen requisitos específicos para el hardware.
